\chapter{This is the first appendix}
By default, LaTeX handles appendices well. However, for the specific formatting of UF, we have to do a bit more. The default behavior is to simply list\begin{center}
    `A THIS IS THE FIRST APPENDIX ...'
\end{center}
in the appendix. If there are more appendices, it will enumerate them with the alphabet (which is partly what UF wants) and list them as though they are chapters.

But, UF has other (read wrong) opinions. If there is a single appendix, it should be titled as:
\begin{center}
    `APPENDIX \{APPENDIX TITLE\}'
\end{center}
with ToC entry as
\begin{table}[]
    \centering
    \begin{tabular}{l l c r}
        APPENDIX& THIS IS THE FIRST APPENDIX TITLE& ... & PAGE\\
    \end{tabular}
    \caption[Example toc for single appendix]{Example toc for single appendix}
    \label{tab:AppendixToC}
\end{table}
with figures and the such included into their respective List of {\*} as \{object\} \{A-$x$\}
If you have more appendices, then the behavior should change. See below.
